# DMECS-YOLOv10

## Installation
`conda` virtual environment is recommended. 
```
conda create -n dmecs python=3.9
conda activate dmecs
pip install -r requirements.txt
pip install -e .
```

## Training 
please use the "myTrain.py" to train your own datasets.
